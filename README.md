CCCInventory is an order management system for a bakery. It replaces paper order forms with a digital system for tracking custom cake, cupcake, cookie, and pupcake orders.

## Features

- **Order management** — Create, edit, and soft-delete orders; track customer info, delivery details, and pricing
- **Bake sheet** — Weekly summary of all items to bake, grouped and sorted by type; collapsible order list for day-of use; printable
- **Calendar** — Month and week view of upcoming orders with pickup/delivery color coding
- **Management page** — Maintain option lists (flavors, sizes, shapes) and signature cupcake presets; restore soft-deleted orders
- **Attachments** — Upload and manage files attached to individual orders
- **Invoice tracking** — Track deposit and final payment amounts, methods, and dates

## Tech Stack

- C# .NET 10 / ASP.NET Core
- Entity Framework Core (SQLite)
- Angular 19 (standalone components)
- Bootstrap 5
- ng-bootstrap
