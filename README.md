CCCInventory (WIP) is order management system for my wife's bakery. We currently take in orders on paper forms, and thought it would be good to digitize. 

Features:
- Custom ordering system, allowing you to enter custom cake and cupcake orders, notes, delivery instructions, and price
- Edit and modify any part of existing orders
- Tracks paid/unpaid invoices
- Generates a dynamic bake sheet so the baker can see at a glance what needs to be made in the upcoming days/week
- Generates a calendar showing what orders are on each day of the week

Tech Stack:
- C# .NET 8 Core
- Entity Framework Core
- Angular 17
- Bootstrap 5
- SQL Server
